# Example Configuration Files

This directory contains example configuration files for different environments:

- `mcp_config.json`: Default MCP server configuration for Docker
- `mcp_config_local.json`: Local development MCP server configuration
- `mcp_config_updated.json`: Updated MCP server configuration
- `vscode_config.json`: VS Code extension configuration
- `claude_config.json`: Claude AI configuration

To use these files, copy the appropriate configuration file to your project and rename it as needed.
